AAVAIL Market Analysis Presentation

View the presentation using one of the methods below:

Method 1: RISE Slideshow (Recommended - Interactive)

For the best experience with interactive slideshow features:

  1. Open Jupyter Notebook: 
    jupyter notebook notebooks/aavail_presentation.ipynb
  2. Run all cells: Cell > Run All
  3. Click the RISE button in the toolbar (or press Alt+R)
  4. Navigate with arrow keys or space bar
Note: Requires RISE extension installed. If not installed: 
pip install RISE
jupyter-nbextension install rise --py --sys-prefix
jupyter-nbextension enable rise --py --sys-prefix

Method 2: HTML Export (Static - Browser Viewable)

View as a static HTML page in any browser:

  1. Open the exported HTML file: 
    open reports/aavail_presentation.html
    Or navigate to: reports/aavail_presentation.html
  2. All visualizations and content will be visible

Troubleshooting

If RISE button doesn't appear:
  • Verify RISE is installed: pip list | grep -i rise
  • Check extension is enabled: jupyter-nbextension list
  • Restart Jupyter Notebook after installation
If images don't display:
  • Ensure all cells have been run
  • Check that reports/assets/ contains all PNG files
  • Re-run the notebook from the beginning
